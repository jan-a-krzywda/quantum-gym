# Quantum Gym Readout

This repository contains quantum data scripts under `quantum_code/` and ML workflow scripts under `ML/`.

## How to run code

Run all commands from the repository root:

```bash
cd /Users/krzywdaja/Documents/quantum-gym
```

### 1) Train the VAE model

```bash
python ML/training/train_vae_model.py
```

### 2) Generate latent dynamics report

```bash
python ML/analysis_processing/generate_latent_dynamics.py
```

### 3) Generate synthetic zebra data

```bash
python ML/analysis_processing/generate_synthetic_zebra.py --latent-mode iid
```

### 4) Simulate latent zebra from fitted dynamics

```bash
python ML/analysis_processing/simulate_latent_zebra.py --sim-mode ou
```

### 5) Plot latent outputs

```bash
python ML/plotting/plot_latent_dynamics.py
python ML/plotting/plot_latent_from_checkpoint.py
```

## Notes

- If you use a virtual environment, activate it before running the commands.
- Some scripts require model checkpoints and input data generated by previous steps.
